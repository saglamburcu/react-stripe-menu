## Installation
First clone the project.
```
git clone https://github.com/saglamburcu/react-stripe-menu.git
```

## Usage
After cloning the project, open it in Visual Studio Code.
```
cd react-stripe-menu
code .
```
Download node modules to make it work
```
npm install
```
To make it work
```
npm start
```
